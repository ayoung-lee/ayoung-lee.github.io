What is Javascript and its relationship to HTML and CSS?

A carpenter needs a saw to make a table, a developer needs a few programmes to make softwares. Javascript, HTML and CSS together make a good team to make a software. Javascript is a trained carpenter who can choose what material to use, how to use the saw to cut the panels, and decides which part goes where to be assembled as a table. The HTML is each part of the table yet to be assembled. The pieces of legs or table top themselves are a visible sturcture of the table but html itself does not know how to assemble them together. Lastly, CSS is responsible for the outer looks of the table such as whether the tbales is lacquerd or stained in colours. Likewise, the CSS in software decorate what the users can see.

What is DOM

The DOM is abbreciated for Document Object Model. The DOM enables us to interact with the objects in HTML documents. Using the Javascript, we can add, manipulate, remove or change the contents or styling of the documents. In order to do this, we need a specific ids or classes so the computer will know which part we wants to change.

Functions

Functions in Javascript are very essential building blocks. According to mdc web docs, it is described as a procedure. The meaning of which is (n) 1. a series of actions conducted in a certain order or manner. 2. a surgical operation. For example, if you have a robot that runs your errands to buy a packet of milk from countdown, you will have to specifically tell the robot how to get there, what to buy, the quanity of milk you want to buy etc. You can just type in the lines of code to make sure the robot understands what you want. What happens if you want to get more milk? you cannot write down the codes each time you want to change something because that would be a waste of time. Instead, you can use function. First, you would name the function called 'buyMilk' and include all your codes in the function. Therefore, instead of typing all the lines of code, you can simply call this function buyMilk().
