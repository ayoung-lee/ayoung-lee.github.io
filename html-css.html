Margin, Border, and Padding in CSS


In CSS, there are important fundamentals called margin, border, and padding.
In this blog article, I have writen my findings of what they are and how they are used in CSS.


The margin is the space around the element's border.
It can make the gap between the other elements either big or small.
The padding is the space around the contents.
It controls the internal space of an element while the margin controls the outside of an element.
The border is inbetween the margin and padding.

The '.main' is the style of the main class. The heading in the background is effected by this style.
Secondly, the '.box' means for class named box. In the CSS we can assign different value towards top, left, bottom, and right margins.
The units in CSS is px, em, or %. In the current picutre, margin-left is 500px. It means the gap betweeb the box and the other element is 500px.
In this case, the other element is the left end of the page. If there was an other element next to the box and is also set as margin-right:500px,
the total gap between the elements will be 1000px.

The width and height of the elements is simply the size of the object or elements.
The text-align: center mean to make the text go in the center of the relative element.
The padding control the gap between the content and the border. Hence, if the padding increases, the gap between them increases and vice versa.
